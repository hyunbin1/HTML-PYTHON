# 속성 = global attribute

1. 링크 =  
1. lang = 요소의 언어를 정의하는데 도움을 준다. 
2. class = 하나의 반복되는 스타일에 적용하여 정의를 해줌
3. id = 반복되지 않고 세부적인 내용을 정의하기 위해 사용함(로고, 상단메뉴, 하단 정보 등등)
4.  your name here = 브라우저를 사용하는 사람이 직접 내영을 수정할 수 있다.

dir = 텍스트 방향을 지정하는 속성이다
dir="ltr" - 택스트를 왼쪽에서 오른쮹으로 표현한다
dir-"rtl" = 텍스트를 오른쪽에서 왼쪽으로 표현한다
dir="auto" =  브라우저의 설정값에 따른다. 

 
잘 모르겠는 것 : ARIA role -> 1. aria-label, 2. aria-hidden="true" -> 시각장애인을 위해 한 단어로 화면 리더기가 읽을 수 있게 하는 기능?
